├─ 🔧 FUNCIONES DE REVERBERACIÓN
│  │
│  ├─ calculate_reverberation_sabine.m
│  │  ├─ T₆₀ = 0.162 × V / (α_m·S + 4mV)
│  │  ├─ Método clásico, válido para α_m < 0.3
│  │  ├─ Entrada: (alpha_m, S, m, V)
│  │  ├─ Salida: T₆₀ en segundos
│  │  └─ Uso: T = calculate_reverberation_sabine(0.20, 100, 0.000007, 250)
│  │
│  ├─ calculate_reverberation_eyring.m
│  │  ├─ T₆₀ = 0.162 × V / (-S·ln(1-α_m) + 4mV)
│  │  ├─ Mejor para absorción no uniforme
│  │  ├─ Recomendado para α_m > 0.3
│  │  ├─ Entrada: (alpha_m, S, m, V)
│  │  ├─ Salida: T₆₀ en segundos
│  │  └─ Uso: T = calculate_reverberation_eyring(0.25, 100, 0.000007, 250)
│  │
│  └─ calculate_reverberation_millington.m
│     ├─ T₆₀ = 0.162 × V / (Σ[-S_i·ln(1-α_i)] + 4mV)
│     ├─ Óptimo para materiales heterogéneos
│     ├─ Especifica absorción de cada superficie
│     ├─ Entrada: (superficies_matriz, S, m, V)
│     ├─ Salida: T₆₀ en segundos
│     └─ Uso: T = calculate_reverberation_millington([80,0.05; 20,0.80], S, m, V)
│
│
├─ 🔊 FUNCIONES DE IMPACTO ACÚSTICO
│  │
│  ├─ calculate_impact_level.m
│  │  ├─ L_i = 10·log₁₀(1/n Σ 10^(L_j/10))
│  │  ├─ Promedio energético de mediciones
│  │  ├─ Entrada: matriz [posiciones × frecuencias] en dB
│  │  ├─ Salida: vector de niveles promediados
│  │  └─ Uso: L_i = calculate_impact_level(L_measurements)
│  │
│  ├─ calculate_standardized_impact_level.m
│  │  ├─ L'_nT = L_i - 10·log₁₀(T/T₀)
│  │  ├─ Corrección por reverberación (ISO 16283-2)
│  │  ├─ Normalización según recinto
│  │  ├─ Entrada: (L_i, T, T0)
│  │  ├─ Salida: L'_nT en cada frecuencia
│  │  └─ Uso: L_nT = calculate_standardized_impact_level(L_i, T, 0.5)
│  │
│  └─ calculate_weighted_impact_level.m
│     ├─ Método ISO 717-2
│     ├─ Desplazamiento de curva de referencia
│     ├─ Comparación y ajuste automático
│     ├─ Entrada: (frecuencias, L_nT)
│     ├─ Salida: L'_nT,w valor único en dB
│     └─ Uso: L_nTw = calculate_weighted_impact_level(freq, L_nT)
│
