🔴 FUNCIONES MODULARES - REVERBERACIÓN (3)
───────────────────────────────────────────

3. calculate_reverberation_sabine.m (30 líneas)
   ├─ Fórmula: T₆₀ = 0.162 × V / (α_m·S + 4mV)
   ├─ Válido para absorción homogénea (α_m < 0.3)
   ├─ Entrada: (alpha_m, S, m, V)
   ├─ Salida: T₆₀ en segundos
   ├─ Documentación técnica incluida
   └─ Uso: T = calculate_reverberation_sabine(0.20, 100, 0.000007, 250)

4. calculate_reverberation_eyring.m (35 líneas)
   ├─ Fórmula: T₆₀ = 0.162 × V / (-S·ln(1-α_m) + 4mV)
   ├─ Mejor para absorción no uniforme (α_m > 0.3)
   ├─ Entrada: (alpha_m, S, m, V)
   ├─ Salida: T₆₀ en segundos
   ├─ Validación automática de parámetros
   └─ Uso: T = calculate_reverberation_eyring(0.25, 100, 0.000007, 250)

5. calculate_reverberation_millington.m (50 líneas)
   ├─ Fórmula: T₆₀ = 0.162 × V / (Σ[-S_i·ln(1-α_i)] + 4mV)
   ├─ Óptimo para materiales heterogéneos específicos
   ├─ Entrada: (matriz superficies, S, m, V)
   ├─ Salida: T₆₀ en segundos
   ├─ Manejo automático de múltiples materiales
   └─ Uso: T = calculate_reverberation_millington([80,0.05; 20,0.8], S, m, V)


🔴 FUNCIONES MODULARES - IMPACTO ACÚSTICO (3)
──────────────────────────────────────────────

6. calculate_impact_level.m (30 líneas)
   ├─ Fórmula: L_i = 10·log₁₀(1/n Σ 10^(L_j/10))
   ├─ Promedio energético de mediciones ISO 16283-2
   ├─ Entrada: Matriz [posiciones × frecuencias]
   ├─ Salida: Vector de niveles promediados por frecuencia
   ├─ Procesamiento vectorizado en MATLAB
   └─ Uso: L_i = calculate_impact_level(L_measurements)

7. calculate_standardized_impact_level.m (35 líneas)
   ├─ Fórmula: L'_nT = L_i - 10·log₁₀(T/T₀)
   ├─ Corrección por reverberación del recinto (ISO 16283-2)
   ├─ Normalización a reverberación de referencia (T₀ = 0.5s)
   ├─ Entrada: (L_i, T, T0)
   ├─ Salida: L'_nT en cada banda de frecuencia
   ├─ Validación automática de parámetros
   └─ Uso: L_nT = calculate_standardized_impact_level(L_i, T_eyring, 0.5)

8. calculate_weighted_impact_level.m (100 líneas)
   ├─ Método ISO 717-2: Desplazamiento de curva de referencia
   ├─ Comparación automática con curva normalizada
   ├─ Cálculo de desviaciones desfavorables (UNFAV)
   ├─ Ajuste iterativo de posición (incrementos de 1 dB)
   ├─ Conversión a valor único L'_nT,w @ 500 Hz
   ├─ Entrada: (frecuencias, L'_nT)
   ├─ Salida: L'_nT,w valor único en dB
   ├─ Información de depuración en consola
   └─ Uso: L_nTw = calculate_weighted_impact_level(freq, L_nT)
